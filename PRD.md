## 1. 产品概述

### 1.1 产品定位

一款基于 Cloudflare Pages + Remove.bg API 的轻量级图片去背景在线工具，为用户提供快速、便捷、高质量的图片背景移除服务。

### 1.2 目标用户

| 用户类型 | 使用场景 | 核心需求 |
|---------|---------|---------|
| 电商运营 | 商品图去背用于详情页/主图 | 批量处理、高质量输出 |
| 自媒体创作者 | 制作封面图、素材合成 | 快速出图、简单易用 |
| 普通用户 | 个人照片处理、证件照换底 | 免费额度、操作简单 |
| 设计师 | 快速抠图辅助设计工作 | 透明背景、高清下载 |

### 1.3 核心价值主张

- ⚡ **快速**: 3-5 秒完成处理，无需注册即可使用
- 🎯 **精准**: 依托 Remove.bg AI 模型，抠图精度高
- 🔒 **安全**: 纯内存处理，图片不存储、不泄露
- 💰 **低成本**: 基于 Cloudflare 免费额度 + Remove.bg 免费 50 张/月

## 3. 功能详细设计

### 3.1 图片上传模块

**用户故事**: 作为用户，我希望快速上传图片，以便进行去背景处理。

**功能流程**:
```
用户进入页面 → 拖拽/点击上传 → 文件格式校验 → 文件大小校验 → 读取预览
```

**交互细节**:
- 拖拽区域高亮反馈（dragover 状态）
- 上传过程中显示文件名称和大小
- 格式错误时提示「请上传图片文件（JPG、PNG、WebP）」
- 大小超限提示「图片大小不能超过 10MB」

**技术实现**:
- HTML5 File API 读取文件
- FileReader 转换为 DataURL 预览
- 前端校验文件类型和大小

---

### 3.2 去背景处理模块

**用户故事**: 作为用户，我希望一键去除图片背景，以便获得透明背景图片。

**功能流程**:
```
点击处理按钮 → 显示加载动画 → 调用 API → 接收结果 → 展示对比
```

**API 调用**:
```javascript
POST /api/remove-bg
Content-Type: multipart/form-data

Parameters:
- image_file: File (图片文件)
- size: "auto"

Response:
- 200: image/png (处理后的图片)
- 400: 参数错误
- 403: API Key 无效
- 500: 服务器错误
```

**加载状态**:
- 显示旋转动画
- 文案：「正在处理中... 通常需要 3-5 秒」
- 禁止重复点击

**错误处理**:
| 错误码 | 用户提示 | 处理方案 |
|-------|---------|---------|
| 400 | 「图片格式不支持，请重试」 | 引导重新上传 |
| 403 | 「服务配置错误，请联系管理员」 | 检查 API Key |
| 500 | 「处理失败，请检查网络连接后重试」 | 显示重试按钮 |
| 超时 | 「处理超时，请重试或更换图片」 | 默认超时 30s |

---

### 3.3 结果展示模块

**用户故事**: 作为用户，我希望清晰对比原图和处理结果，以便确认效果。

**布局设计**:
```
┌─────────────────────────────────────┐
│  原图              去背后           │
│ ┌─────────┐      ┌─────────┐       │
│ │         │      │         │       │
│ │  图片   │      │  图片   │       │
│ │         │      │         │       │
│ └─────────┘      └─────────┘       │
└─────────────────────────────────────┘
```

**交互细节**:
- 双图并排展示（移动端上下排列）
- 背景使用棋盘格图案表示透明
- 图片居中显示，保持原始比例
- 最大高度限制，支持点击放大（二期）

---

### 3.4 背景色切换模块

**用户故事**: 作为用户，我希望更换背景色预览效果，以便选择最合适的方案。

**功能设计**:
- 预设 7 种背景选项：透明、白、黑、红、绿、蓝、黄
- 点击按钮即时切换预览
- 当前选中项高亮显示（蓝色边框）
- 默认选中「透明」

**技术实现**:
- CSS background 属性控制
- 下载时根据选择合成背景

---

### 3.5 图片下载模块

**用户故事**: 作为用户，我希望下载处理后的图片，以便在其他地方使用。

**下载选项**:

| 按钮 | 功能 | 输出格式 |
|-----|------|---------|
| 下载透明 PNG | 下载纯去背图片 | PNG（透明背景） |
| 下载带背景图 | 下载当前预览效果的图片 | PNG（含选中背景色） |

**文件命名**:
- 透明 PNG: `background-removed.png`
- 带背景：`image-with-background.png`

**技术实现**:
- 透明 PNG: 直接下载 API 返回的 Blob
- 带背景：Canvas 合成后导出

## 5. 合规与风险控制

### 5.1 数据安全

| 风险点 | 控制措施 |
|-------|---------|
| 用户图片泄露 | 纯内存处理，请求结束后自动销毁，不存储 |
| API Key 泄露 | 存储在 Cloudflare 环境变量，前端不可见 |
| 传输安全 | 全站 HTTPS，Cloudflare 自动证书 |

### 5.2 内容合规

| 风险点 | 控制措施 |
|-------|---------|
| 用户上传违规图片 | 页面明确免责声明，禁止上传敏感/侵权图片 |
| 版权风险 | 免责声明提示用户自行负责图片版权 |
| 滥用风险 | 二期添加 IP 限流、验证码 |

### 5.3 免责声明文案

```
⚠️ 免责声明：请勿上传敏感、侵权或违规图片。
图片仅在内存中处理，请求结束后自动销毁。
本服务仅供学习和个人使用。
```
